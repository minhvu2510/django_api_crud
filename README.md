## Introduction
```
- Backend (django rest framework)
  - git clone https://github.com/minhvu2510/django_api_crud.git'
  - Install venv Python 3.8.2
  - Pip install -r rquirements.txt
  - python manage.py runserver

- Frontend
  - npm run dev (For dev)
  - npm run build (For pro)
```
